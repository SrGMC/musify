Musify is a simple and fast music player, built from the ground up, using web technologies such as HTML, CSS and JavaScript, all wrapped on Electron Framework, currently available for macOS

Features

Musify is a simple music player that features:

  • Night and day modes
  • Loop mode
  • Notifications on song change

Issues

  • Shuffle mode is not implemented yet.

As this is my first project as a computer science student, I've tried my best to write a code that follows common coding conventions. If you encounter any bugs or issues, feel free to contribute to the project. More information can be found here.
