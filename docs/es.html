Musify es un reproductor de música simple y rápido, construido desde cero, que utiliza tecnologías web como HTML, CSS y JavaScript y Electron, disponible para macOS.

Útlimas versiones

Prestaciones

Musify es un simple reproductor de música con las siguientes funciones:

  • Modos de día y noche
  • Repetición
  • Notificaciones al cambiar de canción

Problemas

  • El modo aleatorio aún no está implementado.

Como este es mi primer proyecto como estudiante de ingeniería informática, he intentado escribir un código que sigue convenciones comunes. Si encuentra cualquier bug o fallo, no dude en reportarlo y contribuir al proyecto. Puedes encontrar más información aquí.
